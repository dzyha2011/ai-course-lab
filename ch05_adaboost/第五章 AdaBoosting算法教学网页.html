🎯 AdaBoosting核心思想

集成学习的智慧：让弱者变强

AdaBoosting通过组合多个"弱分类器"，构建出性能优异的"强分类器"。就像一个团队中，每个成员都有自己的专长，通过合理的分工和协作，整个团队的能力远超个体。

🔄 自适应学习机制

核心理念：从错误中学习，让模型更加关注难以分类的样本

  • 样本权重调整：错误分类的样本获得更高权重，下一轮训练时更受关注
  • 分类器权重：性能好的分类器在最终决策中拥有更大话语权
  • 迭代优化：每一轮都在前一轮的基础上改进，逐步提升整体性能

📊 数学原理详解

1. 样本权重初始化

w₁,ᵢ = 1/N, i = 1,2,...,N

其中 N 为训练样本总数，所有样本初始权重相等

2. 分类错误率计算

εₜ = Σᵢ wₜ,ᵢ · I(hₜ(xᵢ) ≠ yᵢ) / Σᵢ wₜ,ᵢ

I(·) 为指示函数，当分类错误时为1，正确时为0

3. 分类器权重计算

αₜ = (1/2) × ln((1-εₜ)/εₜ)

错误率越低，分类器权重越大；错误率接近0.5时，权重接近0

4. 样本权重更新

wₜ₊₁,ᵢ = wₜ,ᵢ × exp(-αₜ × yᵢ × hₜ(xᵢ)) / Zₜ

Zₜ 为归一化因子，确保权重和为1

5. 最终强分类器

H(x) = sign(Σₜ αₜ × hₜ(x))

所有弱分类器的加权投票结果

# 导入必要的库
from sklearn.ensemble import AdaBoostClassifier
from sklearn.model_selection import train_test_split, cross_val_score
from sklearn.metrics import accuracy_score, confusion_matrix, classification_report
from sklearn.datasets import load_breast_cancer
import matplotlib.pyplot as plt
import numpy as np
import pandas as pd

# 设置中文字体
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

print("🎯 AdaBoosting算法实验开始")
print("=" * 50)

# 1. 加载数据集
print("📊 加载乳腺癌数据集...")
data = load_breast_cancer()
X = data.data  # 特征矩阵 (569, 30)
y = data.target  # 标签 (569,)
feature_names = data.feature_names

print(f"数据集信息：")
print(f"  - 样本数量: {X.shape[0]}")
print(f"  - 特征数量: {X.shape[1]}")
print(f"  - 类别分布: 恶性={np.sum(y==0)}, 良性={np.sum(y==1)}")

# 2. 数据分割
print("\n🔄 分割训练集和测试集...")
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42, stratify=y
)

print(f"训练集大小: {X_train.shape[0]}")
print(f"测试集大小: {X_test.shape[0]}")

# 3. 创建和训练AdaBoost模型
print("\n🚀 创建AdaBoost模型...")
model = AdaBoostClassifier(
    n_estimators=50,  # 弱分类器数量
    learning_rate=1.0,  # 学习率
    random_state=42
)

print("⏳ 训练模型中...")
model.fit(X_train, y_train)
print("✅ 模型训练完成！")

# 4. 模型预测和评估
print("\n📈 模型评估结果：")
print("=" * 30)

# 训练集预测
y_train_pred = model.predict(X_train)
train_accuracy = accuracy_score(y_train, y_train_pred)
print(f"训练集准确率: {train_accuracy:.4f}")

# 测试集预测
y_test_pred = model.predict(X_test)
test_accuracy = accuracy_score(y_test, y_test_pred)
print(f"测试集准确率: {test_accuracy:.4f}")

# 交叉验证
cv_scores = cross_val_score(model, X_train, y_train, cv=5)
print(f"5折交叉验证平均准确率: {cv_scores.mean():.4f} (±{cv_scores.std()*2:.4f})")

# 5. 详细分类报告
print("\n📋 详细分类报告：")
print(classification_report(y_test, y_test_pred, 
                          target_names=['恶性', '良性']))

# 6. 混淆矩阵
print("\n🎯 混淆矩阵：")
cm = confusion_matrix(y_test, y_test_pred)
print(cm)
print("\n混淆矩阵解读：")
print(f"  真阴性(TN): {cm[0,0]} - 正确预测为恶性")
print(f"  假阳性(FP): {cm[0,1]} - 错误预测为良性")
print(f"  假阴性(FN): {cm[1,0]} - 错误预测为恶性")
print(f"  真阳性(TP): {cm[1,1]} - 正确预测为良性")

# 7. 特征重要性分析
print("\n🔍 特征重要性分析（前10个特征）：")
feature_importance = model.feature_importances_
important_features = np.argsort(feature_importance)[-10:][::-1]

for i, idx in enumerate(important_features):
    print(f"{i+1:2d}. {feature_names[idx]:<25} {feature_importance[idx]:.4f}")

print("\n🎉 实验完成！")

# 弱分类器数量对性能的影响分析
print("\n📊 分析弱分类器数量对性能的影响...")

# 测试不同的n_estimators值
n_estimators_range = [1, 5, 10, 20, 30, 50, 70, 100]
train_scores = []
test_scores = []

for n_est in n_estimators_range:
    # 创建模型
    ada_model = AdaBoostClassifier(n_estimators=n_est, random_state=42)
    
    # 训练模型
    ada_model.fit(X_train, y_train)
    
    # 计算准确率
    train_score = ada_model.score(X_train, y_train)
    test_score = ada_model.score(X_test, y_test)
    
    train_scores.append(train_score)
    test_scores.append(test_score)
    
    print(f"n_estimators={n_est:3d}: 训练={train_score:.4f}, 测试={test_score:.4f}")

# 可视化结果
plt.figure(figsize=(10, 6))
plt.plot(n_estimators_range, train_scores, 'o-', label='训练集准确率', linewidth=2)
plt.plot(n_estimators_range, test_scores, 's-', label='测试集准确率', linewidth=2)
plt.xlabel('弱分类器数量 (n_estimators)')
plt.ylabel('准确率')
plt.title('AdaBoost性能随弱分类器数量变化')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

print(f"\n🎯 最佳弱分类器数量: {n_estimators_range[np.argmax(test_scores)]}")
print(f"🏆 最高测试准确率: {max(test_scores):.4f}")

from sklearn.naive_bayes import MultinomialNB

# 创建数据集
df_positive = pd.DataFrame({"comment": positive_comments, "label": 1})
df_negative = pd.DataFrame({"comment": negative_comments, "label": 0})
df = pd.concat([df_positive, df_negative], ignore_index=True)

print(f"数据集信息：")
print(f"  - 总样本数: {len(df)}")
print(f"  - 正面评论: {len(df_positive)}")
print(f"  - 负面评论: {len(df_negative)}")

# 2. 文本预处理
print("\n🔧 文本预处理...")

def preprocess_text(text):
    """文本预处理函数"""
    # 使用jieba进行中文分词
    words = jieba.cut(text)
    # 过滤停用词（简化版）
    stop_words = {'的', '了', '在', '是', '我', '有', '和', '就', '不', '人', '都', '一', '一个', '上', '也', '很', '到', '说', '要', '去', '你', '会', '着', '没有', '看', '好', '自己', '这'}
    words = [word for word in words if word not in stop_words and len(word) > 1]
    return " ".join(words)

# 对评论进行预处理
df["processed_comment"] = df["comment"].apply(preprocess_text)

print("预处理示例：")
for i in range(2):
    print(f"原文: {df.iloc[i]['comment']}")
    print(f"处理后: {df.iloc[i]['processed_comment']}")
    print()

# 3. 数据分割
print("🔄 分割训练集和测试集...")
X = df["processed_comment"]
y = df["label"]

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42, stratify=y
)

print(f"训练集大小: {len(X_train)}")
print(f"测试集大小: {len(X_test)}")

# 4. 构建模型管道
print("\n🚀 构建AdaBoost情感分析模型...")

# 创建包含特征提取和分类的管道
model = make_pipeline(
    TfidfVectorizer(
        max_features=1000,  # 最大特征数
        ngram_range=(1, 2),  # 使用1-gram和2-gram
        min_df=1,  # 最小文档频率
        max_df=0.9  # 最大文档频率
    ),
    AdaBoostClassifier(
        base_estimator=MultinomialNB(),  # 使用朴素贝叶斯作为弱分类器
        n_estimators=50,
        learning_rate=1.0,
        random_state=42
    )
)

# 5. 训练模型
print("⏳ 训练模型中...")
model.fit(X_train, y_train)
print("✅ 模型训练完成！")

# 6. 模型评估
print("\n📈 模型评估结果：")
print("=" * 30)

# 预测
y_train_pred = model.predict(X_train)

# 7. 实际应用函数
def predict_sentiment(text, show_probability=True):
    """预测单条评论的情感倾向"""
    # 预处理
    processed_text = preprocess_text(text)
    
    # 预测
    prediction = model.predict([processed_text])[0]
    probability = model.predict_proba([processed_text])[0]
    
    sentiment = "正面" if prediction == 1 else "负面"
    confidence = max(probability)
    
    result = f"评论: {text}\n"
    result += f"情感倾向: {sentiment}\n"
    if show_probability:
        result += f"置信度: {confidence:.2f}\n"
        result += f"详细概率: 负面={probability[0]:.2f}, 正面={probability[1]:.2f}"
    
    return result

# 9. 批量分析功能
def batch_sentiment_analysis(comments):
    """批量情感分析"""
    results = []
    for comment in comments:
        processed = preprocess_text(comment)
        pred = model.predict([processed])[0]
        prob = model.predict_proba([processed])[0][1]
        
        results.append({
            'comment': comment,
            'sentiment': '正面' if pred == 1 else '负面',
            'positive_prob': prob,
            'confidence': max(prob, 1-prob)
        })
    
    return pd.DataFrame(results)
