局部线性嵌入(LLE)实验教学

探索非线性降维算法的原理与应用

算法原理

🎯 核心思想

LLE假设高维数据点在局部区域内呈现线性结构，每个数据点可以通过其近邻点的线性组合来重构。通过保持这种局部线性重构关系，实现将高维流形展开到低维空间。

🔍 算法优势

  • 无需迭代优化，计算复杂度适中
  • 能有效展开非线性流形结构
  • 保留数据局部几何特性
  • 适合处理瑞士卷、人脸图像等复杂数据

⚠️ 算法局限

  • 对噪声和离群点敏感
  • 无法直接处理样本外点
  • 参数选择对结果影响较大
  • 在稀疏高维数据上表现不佳

算法步骤

1
近邻选择
对每个数据点，根据欧氏距离找到其k个最近邻点。近邻数量k的选择至关重要：k过小会导致流形撕裂，k过大会失去局部性。
2
权重计算
通过最小化重构误差求解权重矩阵W，使得每个数据点能被其近邻点线性表示。约束条件包括非近邻点权重为0和权重归一化。
3
低维嵌入
在低维空间中寻找数据表示Y，保持权重矩阵W不变。通过特征值分解求解，取最小的d+1个特征值对应的特征向量作为低维嵌入结果。

📐 数学推导

步骤1：寻找K近邻

对于每个数据点 xi，找到其K个最近邻居：

N(i) = {j : ||xi - xj|| 是前K小的}

步骤2：计算重构权重

最小化重构误差，求解权重矩阵 W：

ε(W) = Σi ||xi - Σj∈N(i) Wijxj||²

约束条件：

  • Wij = 0 如果 j ∉ N(i) （非邻居权重为0）
  • Σj Wij = 1 （权重归一化）

权重计算的闭式解

对于每个点 i，权重向量 wi 的闭式解为：

wi = Gi-11 / (1TGi-11)

其中 Gi 是局部Gram矩阵：

Gi(j,k) = (xi - xj)T(xi - xk)

步骤3：求解低维嵌入

在低维空间中保持相同的权重关系，最小化：

Φ(Y) = Σi ||yi - Σj Wijyj||²

这等价于求解广义特征值问题：

(I - W)T(I - W)Y = λY

其中 Y 的列向量是对应最小非零特征值的特征向量。

7. 应用分析题
分析LLE在图像检索系统中的优势和局限性，并提出可能的改进方法。

研究改进LLE（Modified LLE）和LTSA（Local Tangent Space Alignment）算法，分析它们如何解决标准LLE的不足。
算法 类型 核心思想 优势场景 时间复杂度
PCA 线性 最大化方差 线性结构数据、去噪 O(ND²)
LLE 非线性 保持局部线性关系 流形结构数据、可视化 O(Nk³ + N²d)

import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import make_swiss_roll
from sklearn.manifold import LocallyLinearEmbedding

# 2. 拟合模型并转换数据（填空2：调用fit_transform方法）
X_lle = lle.fit_transform(X)

# 可视化结果
fig = plt.figure(figsize=(12, 6))

ax1 = fig.add_subplot(121, projection='3d')

ax1.set_title('Original Swiss Roll Data')
ax1.view_init(azim=120, elev=10)

# 3. 绘制LLE降维后2D数据（填空3：补充X_lle的两个维度）

手写数字数据集降维实验 
# 导入必要库
import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import load_digits
from sklearn.manifold import LocallyLinearEmbedding
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score

# 加载手写数字数据集
digits = load_digits()
X, y = digits.data, digits.target
print(f"Dataset shape: {X.shape}, Class distribution: {np.bincount(y)}")

# 数据预处理：归一化
X = X / X.max()

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42
)

# 5. 创建LLE模型（填空5：设置n_neighbors=15, n_components=20）
lle = LocallyLinearEmbedding(n_neighbors=15, n_components=20, random_state=42)

# 6. 拟合训练数据并转换（填空6：调用fit_transform和transform方法）
X_train_lle = lle.fit_transform(X_train)
X_test_lle = lle.transform(X_test)

# 使用KNN进行分类（降维前后对比）
knn_orig = KNeighborsClassifier(n_neighbors=5)
knn_orig.fit(X_train, y_train)
y_pred_orig = knn_orig.predict(X_test)
acc_orig = accuracy_score(y_test, y_pred_orig)

# 7. 在降维数据上训练KNN（填空7：使用X_train_lle训练）
knn_lle = KNeighborsClassifier(n_neighbors=5)
knn_lle.fit(X_train_lle, y_train)
y_pred_lle = knn_lle.predict(X_test_lle)
acc_lle = accuracy_score(y_test, y_pred_lle)

print(f"Original data accuracy: {acc_orig:.4f}")
print(f"LLE reduced data accuracy: {acc_lle:.4f}")
print(f"Feature dimension reduced from {X.shape[1]} to {X_train_lle.shape[1]}")
