🎯 A*算法核心概念

A*算法是一种启发式搜索算法，它巧妙地结合了Dijkstra算法的完备性和贪婪搜索的高效性。

💡 核心思想

通过评估函数引导搜索方向，优先探索最有希望的路径，从而在复杂状态空间中高效找到最优解。

📊 评估函数

f(n) = g(n) + h(n)

• g(n): 从起点到当前节点的实际代价
• h(n): 从当前节点到目标的估计代价（启发函数）
• f(n): 节点的综合评估代价

🔄 算法流程

  1. 初始化：将起点加入开放列表，设置g(n)=0
  2. 选择节点：从开放列表中选择f(n)最小的节点
  3. 目标检测：如果是目标节点，回溯路径并结束
  4. 扩展邻居：生成所有可达的邻接节点
  5. 更新代价：计算邻居节点的g、h、f值
  6. 重复搜索：重复步骤2-5直到找到解或无解

🎯 启发函数类型

启发函数 公式 适用场景 特点
曼哈顿距离 |x₁-x₂| + |y₁-y₂| 网格地图 计算简单，可采纳
欧几里得距离 √[(x₁-x₂)² + (y₁-y₂)²] 连续空间 更精确但计算复杂
对角线距离 max(|x₁-x₂|, |y₁-y₂|) 8方向移动 平衡精度与效率

🏗️ 节点类定义

class AStarNode:
    def __init__(self, state, parent=None, g=0, h=0):
        self.state = state      # 当前状态
        self.parent = parent    # 父节点
        self.g = g             # 实际代价
        self.h = h             # 启发函数值
        self.f = g + h         # 评估函数值
    
    def __lt__(self, other):
        return self.f < other.f

🧮 启发函数实现

def manhattan_distance(state, goal):
    """计算曼哈顿距离"""
    distance = 0
    for i in range(9):
        if state[i] != 0 and state[i] != goal[i]:
            goal_pos = goal.index(state[i])
            current_row, current_col = i // 3, i % 3
            goal_row, goal_col = goal_pos // 3, goal_pos % 3
            distance += abs(current_row - goal_row) + abs(current_col - goal_col)
    return distance

def hamming_distance(state, goal):
    """计算错位棋子数"""
    return sum(s != g and s != 0 for s, g in zip(state, goal))

🔍 A*算法核心

def a_star_search(initial_state, goal_state):
    import heapq
    
    # 初始化开放列表和关闭集合
    open_heap = []
    closed_set = set()
    
    # 创建起始节点
    start_node = AStarNode(initial_state, g=0, 
                          h=manhattan_distance(initial_state, goal_state))
    heapq.heappush(open_heap, start_node)
    
    while open_heap:
        # 选择f值最小的节点
        current = heapq.heappop(open_heap)
        
        # 检查是否到达目标
        if current.state == goal_state:
            return reconstruct_path(current)
        
        closed_set.add(current.state)
        
        # 扩展邻居节点
        for neighbor in get_neighbors(current):
            if neighbor.state in closed_set:
                continue
                
            neighbor.h = manhattan_distance(neighbor.state, goal_state)
            neighbor.f = neighbor.g + neighbor.h
            
            heapq.heappush(open_heap, neighbor)
    
    return None  # 无解

def get_neighbors(node):
    """生成当前节点的所有可能邻接节点"""
    neighbors = []
    state = node.state
    blank_pos = state.index(0)
    row, col = blank_pos // 3, blank_pos % 3
    
    # 空格移动方向：上、下、左、右
    directions = [(-1, 0), (1, 0), (0, -1), (0, 1)]
    
    for dr, dc in directions:
        new_row, new_col = row + dr, col + dc
        if 0 <= new_row < 3 and 0 <= new_col < 3:
            # 计算新位置索引
            new_blank_pos = new_row * 3 + new_col
            # 交换空格与相邻数字
            state_list = list(state)
            state_list[blank_pos], state_list[new_blank_pos] = \
                state_list[new_blank_pos], state_list[blank_pos]
            new_state = tuple(state_list)
            # 创建新节点（g值+1）
            new_node = AStarNode(new_state, parent=node, g=node.g + 1)
            neighbors.append(new_node)
    
    return neighbors

def reconstruct_path(node):
    """回溯路径"""
    path = []
    while node:
        path.append(node)
        node = node.parent
    return path[::-1]  # 反转路径，从起点到目标

# 八数码问题完整求解示例
def solve_puzzle(initial_state, goal_state, heuristic=manhattan_distance):
    """求解八数码问题并输出结果"""
    print("初始状态：")
    print_state(initial_state)
    print("目标状态：")
    print_state(goal_state)
    
    path = a_star_search(initial_state, goal_state)
    
    if not path:
        print("该问题无解！")
        return
    
    print(f"找到最优解，共{len(path)-1}步：")
    for i, node in enumerate(path):
        print(f"第{i}步：")
        print_state(node.state)

def print_state(state):
    """打印3x3棋盘状态"""
    for i in range(3):
        row = []
        for j in range(3):
            num = state[i*3 + j]
            row.append(str(num) if num != 0 else " ")
        print(" ".join(row))
    print()
