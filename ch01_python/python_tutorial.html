🔧 函数与模块

📝 函数定义与调用

函数是可重复使用的代码块，使用def关键字定义。

# 简单函数 def greet(): print("你好，世界！") # 带参数的函数 def greet_person(name): print(f"你好，{name}！") # 带返回值的函数 def add_numbers(a, b): return a + b # 函数调用 greet() # 你好，世界！ greet_person("小明") # 你好，小明！ result = add_numbers(5, 3) # result = 8

🎯 函数参数方式

# 位置参数 def introduce(name, age): print(f"我叫{name}，今年{age}岁") # 默认参数 def greet(name, greeting="你好"): print(f"{greeting}，{name}！") # 可变参数 def sum_all(*args): return sum(args) # 关键字参数 def create_profile(**kwargs): for key, value in kwargs.items(): print(f"{key}: {value}") # 函数调用示例 introduce("小红", 18) greet("小明") greet("小李", "早上好") print(sum_all(1, 2, 3, 4, 5)) # 15 create_profile(name="张三", age=25, city="北京")

# 导入整个模块 import math print(math.pi) # 3.141592653589793 print(math.sqrt(16)) # 4.0 # 导入特定函数 from random import randint, choice print(randint(1, 10)) # 随机整数 print(choice(["A", "B", "C"])) # 随机选择 # 使用别名 import datetime as dt now = dt.datetime.now() print(now.strftime("%Y-%m-%d %H:%M:%S"))

🐾 Anaconda安装与配置

Anaconda是一个Python数据科学平台，包含了Python解释器和众多科学计算包。

💿 安装步骤

  1. 访问Anaconda官网：https://www.anaconda.com/download
  2. 根据操作系统选择对应的安装包下载
  3. 运行安装程序，建议选择默认安装路径
  4. 勾选"Add Anaconda to PATH environment variable"（可选）
  5. 完成安装后，可以通过Anaconda Navigator管理环境

🔢 Anaconda与Python版本对应关系

Anaconda版本 Python版本 发布时间 推荐使用
Anaconda3 2024.02 Python 3.11 2024年2月 ✅ 推荐
Anaconda3 2023.09 Python 3.11 2023年9月 ✅ 推荐
Anaconda3 2023.07 Python 3.10 2023年7月 🔄 稳定
Anaconda3 2022.10 Python 3.9 2022年10月 ⚠️ 较旧

💡 版本选择建议

推荐安装：Anaconda3 2024.02（Python 3.11）- 这是目前最新且稳定的版本，兼容性好，包含最新的科学计算包。

学习环境：如果你是初学者，建议使用Python 3.10或3.11版本，这些版本稳定且有良好的库支持。

🛠️ Anaconda常用命令

# 查看conda版本 conda --version # 查看已安装的包 conda list # 创建新的虚拟环境 conda create --name myenv python=3.11 # 激活虚拟环境 conda activate myenv # 在环境中安装包 conda install numpy pandas matplotlib # 使用pip安装包 pip install requests # 查看所有环境 conda env list # 退出当前环境 conda deactivate # 删除环境 conda env remove --name myenv

🔽 官方Python安装步骤

如果你选择安装官方Python版本：

  1. 访问Python官网：https://www.python.org
  2. 下载适合你操作系统的Python安装包（推荐Python 3.10+）
  3. 运行安装程序，记得勾选"Add Python to PATH"
  4. 安装完成后，打开命令行输入 python --version 验证安装

💡 安装提示

对于数据科学和机器学习初学者，强烈推荐使用Anaconda，因为它预装了NumPy、Pandas、Matplotlib等常用包。对于Web开发或通用编程，可以选择官方Python版本。

🌐 虚拟环境配置

虚拟环境可以为不同项目创建独立的Python环境，避免包冲突。

# 使用venv创建虚拟环境（官方Python） python -m venv myproject # 激活虚拟环境 # Windows: myproject\Scripts\activate # macOS/Linux: source myproject/bin/activate # 安装包 pip install requests numpy pandas # 查看已安装的包 pip list # 生成依赖文件 pip freeze > requirements.txt # 从依赖文件安装包 pip install -r requirements.txt # 退出虚拟环境 deactivate
